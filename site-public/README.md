# Good Game Public Pages

Site estatico para GitHub Pages com:

- suporte
- politica de privacidade
- seguranca e moderacao
- termos de uso
- landing simples para links compartilhados (`?meetup=` e `?venue=`)

Para publicar em GitHub Pages:

1. envie estes arquivos para a branch `main`;
2. habilite Pages com source em `main` e pasta `/ (root)`.
